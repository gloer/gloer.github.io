Hva er HTML og CSS?

HTML

HTML er et markeringsspråk. Vi bruker det til å bygge innhold på nettsider.

CSS

Med CSS utformer vi innholdet vi har laget med HTML. Det vil si at vi bestemmer hvordan innholdet skal se ut. Hvilken farge skal teksten ha? Hvor stor skal skriften være? Hvilken bakgrunnsfarge skal nettsiden ha? Dette er bare et lite sett av eksempler på hva CSS kan brukes til.
